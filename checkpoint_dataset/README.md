## Requirements

### Hardware
- CUDA-compatible GPU (required)
- ~29GB free disk space

### Software
- Python 3.x
- CUDA toolkit and drivers
- Python packages (see requirements.txt):
  - torch (with CUDA support)
  - transformers
  - datasets
  - numpy
  - tqdm

## Performance Reference
- **RTX 5070**: ~27 minutes runtime, ~29GB storage
